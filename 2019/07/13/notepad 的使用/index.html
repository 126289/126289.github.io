Notepad的使用

Veröffentlicht am 2019-07-13 Bearbeitet am 2019-07-14 in 编辑器
对于只是简单的编写代码的人来说，IDE十分的笨重，所以简单的编写还是靠文本编辑器吧！

安装教程

前排提示：我现在安装的版本是7.6.6 32位的，以后可能会有改变。

进入官网下载，根据自己的需要来下载，我因为考虑了便携性，下载了32位的，也就是x86的安装包。

如果有便携性的需求，要在安装过程中勾选图片所选项目。

打开程序的根目录里的/plugins文件夹，里面放置的是notepad++的插件，后面安装插件教程中，要将下载的.dll移动到里面，也可以直接在设置里导入.dll文件。

软件简单配置

先进入程序配置一下内容，打开工具栏的设置，里面选项是要修改的内容。点开首选项，将新建里的编码改成ANSI，其他的看个人喜好了。语言格式设置是主题配置，你也可以修改主题为自己想要的形式，并保存。管理快捷键就是修改一些基本的操作的快捷键形式，以后设置插件快捷键也会用的上。

插件

notepad++的插件虽然不是很多，但是绝对够用，可以在工具栏中的插件中的插件管理中下载插件，也可以直接进插件中心中下载，然后在工具栏中的设置中导入。

MIME Tools

安装notepad++时自带插件，可以实现执行Base64 / Quoted Printable / URL编码和解码。具体方法是选中文字，执行解码，可以自己设置快捷键

Npp Converter

安装notepad++时自带插件，ASCII码<->十六进制转换器的插件。

NppExport

安装notepad++时自带插件，将代码颜色保存导出为其他格式，或者源代码以rtf / html格式复制到剪贴板中，是一个非常方便的插件。

DSpellCheck

拼写检查插件，一些简单的语法错误还是能检查出来的。

Explorer

资源管理器，可以在notepad++中就可以对文件的读取。

Code Alignment

代码对齐改进，可以在行与行之间按照特定的符号对齐整理,如下面的代码所示：

Compare Plugin

一个非常有用的差异插件，用于显示2个文件之间的差异（并排）。

DocMonitor

自动保存，3秒一保存。

Hex Editor

16进制编辑器，兼容性好。

Location Navigate

位置记录插件，记录改变的行数，编辑大文件方便。

NewFileBrowser

可以预览网页编写的情况。

NppAutoIndent

自动缩进模块，写c风格的能用到。

NppCrypt

使用各种块密码、散列算法、随机字符加密/解密。

NppExec

您无需离开Notepad ++即可执行命令或保存的脚本。

Npp Hash Maker

用于计算所选文本的哈希值。

XBrackets Lite

符号补全，支持引号，括号。

3P

强大的自动完成，工具-提示每一个单词，一个代码浏览器，以快速浏览您的代码，一个文件资源管理器，方便地访问您的所有资源。

Markdown Panel

md文件的预览，分栏。

c/c++

网页html

说到速写html，不得不提到一个神级插件emmet了，可以进这个页面下载,添加好后，这个插件需要Python Script安装教程，强烈推荐将快捷键的Expand abbreviation项修改成Tab键。看官网的介绍，学习一下使用方法。NewFileBrowser可以实现实时预览

安装好插件后，按Ctrl+e就可以实时预览了。

总结

notepad++虽然有一些不足之处，但还是非常方便的。sublime text 3才是程序员首选之作，就是有点麻烦，学习成本高，但相应的功能非常强大。以后会出关于安装sublime text 3的教程的。
